# Manhole_project

## Pin Connection from Board 
| Pin Name  | Digital Pin                            |
|-----------|----------------------------------------|
|SCK        | D13                                    |
| MOSI      | D11                                    |
| OK        | D6                                     |
| LEFT      | D10                                    |
| RIGHT     | D8                                     |
| RST       | D12                                    |
| RED       | D0                                     |
| GREEN     | A6                                     |
| BLUE      | A7                                     |
| MQ2       | A2                                     |
| MQ7       | A3                                     |
| MQ06      | A1                                     |
| MQ135     | A0                                     |



### MODULES USED
- DISPLAY 16x2 I2C
- SMOKE SENSOR (MQ02,MQ06,MQ07,MQ135)
- MQ02 (Methane, Butane, LPG, Hydrogen)
- MQ07 (CO, benzene)
- MQ06 (iso-butane, propane, LPG, LNG)
- MQ135 (NH3, NOx, alcohol,CO2)
- NAVIGATION BUTTON
- RGB LED
- GSM
