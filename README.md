# Manhole_project

## Pin Connection from Board 
| Pin Name  | Digital Pin                            |
|-----------|----------------------------------------|

|SCK        | D13                                    |
| MOSI      | D11                                    |
| OK        | D6                                     |
| LEFT      | D7                                     |
| RIGHT     | D8                                     |
| BACK      | D9                                     |
| RST       | D12                                    |
| RED       | A5                                     |
| GREEN     | A6                                     |
| BLUE      | A7                                     |
| MQ3       | A1                                     |
| MQ2       | A2                                     |
| MQ7       | A3                                     |
| MQ135     | A4                                     |




### MODULES USED
- DISPLAY 16x2 I2C
- SMOKE SENSOR (MQ02,MQ03,MQ07,MQ135)
- NAVIGATION BUTTON
- RGB LED
